Question 1

We will replicate the linear regression and regularized linear least squares examples from this week, with a different model. First, we will generate a noisy dataset of 200 samples coming from \[ v_i=u_i^2+\mathcal{N}(0,0.04)\,,\quad i=1,\ldots,200\,, \] where the \(u_i\)â€™s are spaced uniformly in \([-1,1]\), and \(\mathcal{N}(0,0.04)\) means adding Gaussian noise of mean 0 and variance \(0.04\) for each sample.

  • Generate the pairs \((u_i,v_i)\) using suitable random generators. Make a plot illustrating \((u_i,v_i)\). What is the model you can identify to express \(v\) as a function of \(u\)?
Solution 
u <- seq(-1,1,length.out=200)
v <- u^2 + rnorm(200,0, sd=sqrt(0.04))
plot(u,v)

The plot clearly suggests a quadratic relationship between v and u.

  • Describe the least-squares regression problem for finding the optimal parameters in the model \[v(u)=au^2+bu+c\]
Solution

We can write the linear regression model \(v_i=a+bu_i+cu_i^2+\epsilon\) as \[\mathbf{v}= \mathbf{A}\beta+\epsilon\] where \(\beta=(a, b,c)^\top\). If we have \(n\) samples, then \[\mathbf{A}=\begin{pmatrix}1 & u_1 & u_1^2\\ \vdots\\ 1 & u_n & u_n^2\end{pmatrix} \in\mathbb{R}^{n\times 3}\qquad \mathbf{v}=\begin{pmatrix}v_1\\\vdots\\ v_n \end{pmatrix}\in\mathbb{R}^{n}\mbox{ etc.}\]

  • Compute the least squares solution and compute the total least squares error in the \(\ell_2\) norm.
Solution

We can form the normal equations \[\mathbf{A}^\top \mathbf{A}\beta = \mathbf{A}^\top \mathbf{v}\] which we can solve as follows

A <- cbind(rep(1,200), u, u^2)

totalss <- t(v-A%*%betahat)%*%(v-A%*%betahat)
plot(u,v)
u.plt<-seq(-1,1,0.01)

  • Now, instead of solving a regression problem, use the \(v_i\) values to recover a denoised signal using regularized least squares using the same total variation regularization described in the lecture notes
Solution

Here, we assume we are given noisy data \(\mathbf{v}\) on unknown signal \(\mathbf{x}\). To reconstruct \(\mathbf{x}\), we solve the regularized least squares problem \[\min_x \|\mathbf{x}-\mathbf{v}\|_2^2 + \lambda \|L\mathbf{x}\|\] as described in lectures.

We assemble the regularization matrix \(\mathbf{L}\) from the notes

L <- matrix(0, nr=199,nc=200)
for(i in 1:199){
  L[i,i]=1
  L[i,i+1]=-1
}

We set the regularization parameter \(\lambda=100\) ( try different values to see the effect of different regularization values) and find the regularized least squares solution, which we gave in lectures as the solution to

\[(\mathbf{I}+\lambda L^\top L)\mathbf{x}= \mathbf{v}\]

lambda=100
vrecon<-solve(diag(200)+lambda * t(L)%*%L, v)

and we plot the regularized signal against the noisy measurements

plot(u,v)
points(u,vrecon, type='l', col=2)

Question 2

Consider the linear systems \[\begin{align} x+y&=2\\ x+1.001y &=2\nonumber \end{align}\] and \[\begin{align} x+y&=2\\ x+1.001y &=2.001\nonumber \end{align}\]

  • Find the solution of both systems. Check your answer by using the solve command in R.

    Note that a small change in the RHS (\(\mathbf{b}\) in the terminology of the notes) has led to a large change in the solutions (\(\mathbf{x}\)).

Solution 
A <-matrix(c(1,1,1, 1.001), nr=2, byrow=T)
solve(A, c(2,2))
[1] 2 0
solve(A, c(2, 2.001))
[1] 1 1
solve(A, c(2, 2.002))
[1] 0 2
  • We can measure the sensitivity of the linear system \[\mathbf{A}\mathbf{x}=\mathbf{b}\] (where \(\mathbf{A}\) is a positive definite \(n \times n\) matrix) by measuring how much the output \(\mathbf{x}\) changes as \(\mathbf{b}\) changes. Suppose \(\mathbf{b}\) changes to \(\mathbf{b}+\delta\mathbf{b}\), and that this results in the solution \(\mathbf{x}\) changing to \(\tilde{\mathbf{x}}=\mathbf{x}+\delta\mathbf{x}\). Show that the relative change in \(\mathbf{x}\) compared to the relative change to \(\mathbf{b}\) can be expressed as follows: \[\frac{\| \delta \mathbf{x}\| /\|\mathbf{x}\|}{\| \delta\mathbf{b}\| /\|\mathbf{b}\|} = \frac{\| \mathbf{A}^{-1} \delta \mathbf{b}\|}{||\delta \mathbf{b}||}\frac{||\mathbf{A}\mathbf{x}||}{||\mathbf{x}||}\]
Solution

Note that \(\mathbf{A}\mathbf{x}=\mathbf{b}\) and \(\mathbf{A}(\mathbf{x}+\delta\mathbf{x})=\mathbf{b}+\delta\mathbf{b}\) thus \(\mathbf{A}\delta\mathbf{x}=\delta\mathbf{b}\). Thus we see \[\frac{\| \delta \mathbf{x}\| /\|\mathbf{x}\|}{\| \delta\mathbf{b}\| /\|\mathbf{b}\|} = \frac{\| \mathbf{A}^{-1} \delta \mathbf{b}\|}{||\delta \mathbf{b}||}\frac{||\mathbf{A}\mathbf{x}||}{||\mathbf{x}||}.\]

  • The matrix norm induced by the \(L_2\) norm is defined to be \[||\mathbf{A}||_2 = \max \{||\mathbf{A}\mathbf{x}||_2 : ||\mathbf{x}||\leq 1 \}=\max_{\mathbf{x}\not =0} \frac{||\mathbf{A}\mathbf{x}||_2}{||\mathbf{x}||_2} \] Show that
    \[\max_{\mathbf{b}, \delta \mathbf{b}} \frac{\| \delta \mathbf{x}\| /\|\mathbf{x}\|}{\| \delta\mathbf{b}\| /\|\mathbf{b}\|} = ||\mathbf{A}^{-1}||\cdot ||\mathbf{A}||.\]
Solution

Recall that for the induced 2-norm for matrices

\[||\mathbf{A}||_2 = \max \{||\mathbf{A}\mathbf{x}||_2 : ||\mathbf{x}||\leq 1 \}=\max_{\mathbf{x}\not =0} \frac{||\mathbf{A}\mathbf{x}||_2}{||\mathbf{x}||_2} \]

Thus we can see that

\[\max_{\mathbf{b}, \delta \mathbf{b}} \frac{\| \delta \mathbf{x}\| /\|\mathbf{x}\|}{\| \delta\mathbf{b}\| /\|\mathbf{b}\|} = \max_{\delta\mathbf{b}}\frac{\| \mathbf{A}^{-1} \delta \mathbf{b}\|}{||\delta \mathbf{b}||}\max_{\mathbf{x}}\frac{||\mathbf{A}\mathbf{x}||}{||\mathbf{x}||} = ||\mathbf{A}^{-1}||. ||\mathbf{A}||.\]

  • Thus show that for the \(L_2\) norm, we have \[\max_{\mathbf{b}, \delta \mathbf{b}} \frac{\| \delta \mathbf{x}\| /\|\mathbf{x}\|}{\| \delta\mathbf{b}\| /\|\mathbf{b}\|} = \frac{\lambda_{max}(\mathbf{A})}{\lambda_{min}(\mathbf{A})}\] where \(\lambda_{max}(\mathbf{A})\) and \(\lambda_{min}(\mathbf{A})\) are the largest and smallest eigenvalues of \(\mathbf{A}\).

    Hint: The two norm of the matrix is equal to the largest eigenvalue: \[||\mathbf{A}||_2 = \lambda_{max}(\mathbf{A})\]

Solution

Recall that \(||\mathbf{A}||_2 = \lambda_{max}(\mathbf{A})\), and that the eigenvalues of \(\mathbf{A}^{-1}\) are \(1/\lambda_i(\mathbf{A})\) so that \(||\mathbf{A}^{-1}||=\frac{1}{\lambda_{min}(\mathbf{A})}\). The result then follows.
