Question 1

Show that the dot product satisfies the definition of an inner-product.

Solution

We just need to show that \[\langle \mathbf{x}, {\mathbf{y}}\rangle =\mathbf{x}^\top {\mathbf{y}}\] satisfies each of the parts of the definition

  1. Symmetry: \(\langle\mathbf{x}, {\mathbf{y}}\rangle=\langle{\mathbf{y}}, \mathbf{x}\rangle\) for any \(\mathbf{x}, {\mathbf{y}}\in \mathbb{R}^{n}\).

\[\mathbf{x}^\top {\mathbf{y}}= \sum_{i=1}^n x_i y_i = {\mathbf{y}}^\top \mathbf{x}\]

  1. Additivity: \(\langle\mathbf{x}, {\mathbf{y}}+\mathbf{z}\rangle=\langle\mathbf{x}, {\mathbf{y}}\rangle+\langle\mathbf{x}, \mathbf{z}\rangle\) for any \(\mathbf{x}, {\mathbf{y}}, \mathbf{z} \in \mathbb{R}^{n}\).

\[\mathbf{x}^\top ({\mathbf{y}}+\mathbf{z}) = \sum_{i=1}^n x_i (y_i+z+i) = \sum_{i=1}^n x_i y_i + \sum_{i=1}^n x_i +z_i=\mathbf{x}^\top {\mathbf{y}}+\mathbf{x}^\top \mathbf{z}\]

  1. Homogeneity: \(\langle\lambda \mathbf{x}, {\mathbf{y}}\rangle=\lambda\langle\mathbf{x}, {\mathbf{y}}\rangle\) for any \(\lambda \in \mathbb{R}\) and \(\mathbf{x}, {\mathbf{y}}\in \mathbb{R}^{n}\).

  2. Positive definiteness: \(\langle\mathbf{x}, \mathbf{x}\rangle \geq 0\) for any \(\mathbf{x}\in \mathbb{R}^{n}\) and \(\langle\mathbf{x}, \mathbf{x}\rangle=0\) if and only if \(\mathbf{x}=0\).

\[\mathbf{x}^\top \mathbf{x}=\sum_{i=1}^n x_i^2 \geq 0\]

with equality if and only if \(x_i=0\) for all \(i\), i.e., \(\mathbf{x}=\mathbf{0}\).

Question 2

Why is \(\ell_{1 / 2}\) not a norm?

Solution

For it not to be a norm, it must not satisfy one of the three conditions given in the definition. Non-negativity and positivity homogeneity are satisfied, so it must be the triangle inequality that is is not true. To show this, we just need to fine a single counter-example.

Let \(\mathbf{x}=\begin{pmatrix}1\\0\end{pmatrix}\) and \({\mathbf{y}}=\begin{pmatrix}0\\1\end{pmatrix}\) Then \[\|\mathbf{x}\|_{1/2}=\left(\sum \sqrt{x_i}\right)^{2}=1=\|{\mathbf{y}}\|_{1/2}\] but \[\|\mathbf{x}+{\mathbf{y}}\|_{1/2}=(1+1)^{2}=4\] and so we have

\[\|\mathbf{x}+{\mathbf{y}}\| \not<\|\mathbf{x}\|+\|{\mathbf{y}}\|\]

Question 3

Fill in the missing detail near the `why?â€™ in the proof of the Cauchy-Schwartz algorithm.

Solution

We need to prove: \[ \left| \langle \mathbf{x}, {\mathbf{y}}\rangle \right| \leq\|\mathbf{x}\| \cdot\|{\mathbf{y}}\| \] For any \(\lambda \in \mathbb{R}\) : \[\begin{align*} \|\mathbf{x}+\lambda {\mathbf{y}}\|^{2}&=\|\mathbf{x}\|^{2}+2 \lambda\langle\mathbf{x}, {\mathbf{y}}\rangle+\lambda^{2}\|{\mathbf{y}}\|^{2}\\ &=:p(\lambda)\geq 0.\\ \end{align*}\] This is a quadratic in \(\lambda\), which we will call \(p(\lambda)\). We know \(p(\lambda)\geq 0\), and so \(p(\lambda)\) can have at most one real root. The quadratic formula thus tells us that we must have \[4\langle\mathbf{x}, {\mathbf{y}}\rangle^{2}-4\|\mathbf{x}\|^2\|{\mathbf{y}}\|^2\leq 0\] which gives \[ \langle\mathbf{x}, {\mathbf{y}}\rangle^{2}\leq\|\mathbf{x}\|^{2}\|{\mathbf{y}}\|^{2}\,, \] establishing the desired result.

Question 4

Prove \(\|\mathbf{A}\|_{F}\) is a norm on the vector space of matrices?

Solution

Letâ€™s consider the vector space of \(m \times n\) matrices, denoted \(\mathbb{R}^{m\times n}\). We just need to check

  1. Nonnegativity: \[\|\mathbf{A}\|_F=\left(\sum_{ij} a_{ij}^2\right)^{\frac{1}{2}} \geq 0\] for any \(\mathbf{A}\in \mathbb{R}^{m\times n}\) with equality if and only if \(a_{ij}=0\) for all \(i\) and \(j\), i.e., ifF \(\mathbf{A}=\mathbb{0}_{m\times n}\).
  2. Positive homogeneity:

\[\|\lambda \mathbf{A}\|_F=\left(\sum_{ij} \lambda^2 a_{ij}^2\right)^{\frac{1}{2}}=|\lambda|\left(\sum_{ij} a_{ij}^2\right)^{\frac{1}{2}} = |\lambda| \|\mathbf{A}\|_F\]

  1. Triangle inequality: For \(\mathbf{A}= \mathbb{R}^{m\times n}\), denote the columns of \(\mathbf{A}\) as \(\mathbf{a}_1, \ldots, \mathbf{a}_n\), i.e. \[\mathbf{A}= \begin{pmatrix} | & | & \ldots & |\\ \mathbf{a}_1 & \mathbf{a}_2 &\ldots & \mathbf{a}_n\\ | & | & \ldots & | \end{pmatrix}\]

    Then \[\| \mathbf{A}\|_F^2 = \left(\sum_{ij} a_{ij}^2\right) = \sum_{i=1}^n \mathbf{a}_i^\top \mathbf{a}_i\] If we let \[\tilde{\mathbf{a}}=\begin{pmatrix} \mathbf{a}_1\\\mathbf{a}_2\\\vdots\\\mathbf{a}_n\end{pmatrix}\in \mathbb{R}^{mn}\] be the column vector formed by stacking the columns of \(\mathbf{A}\), we can see that \[\|\tilde{\mathbf{a}}\|_2=\left(\tilde{\mathbf{a}}^\top\tilde{\mathbf{a}}\right)^{\frac{1}{2}}=\left(\sum_{i=1}^n \mathbf{a}_i^\top \mathbf{a}_i\right)^{\frac{1}{2}}=\|\mathbf{A}\|_F\] And so, because \(\|\cdot\|_2\) satisifes the triangle inequality, we can conclude that \(\|\cdot \|_F\) does. \[\|\mathbf{A}+\mathbf{B}\|_F = \|\tilde{\mathbf{a}}+\tilde{\mathbf{b}}\|_2 \leq \|\mathbf{a}\|_2+\|\mathbf{b}\|_2=\|\mathbf{A}\|_F+\|\mathbf{B}\|_F\]

Question 5

Prove that \(\operatorname{Tr}(\mathbf{A})=\sum_{i=1}^{n} \lambda_{i}(\mathbf{A})\) (the trace of \(\mathbf{A}\)) and \(\operatorname{det}(\mathbf{A})=\Pi_{i=1}^{n} \lambda_{i}(\mathbf{A})\).

Solution

Because \(\mathbf{A}\) is real and symmetric we can diagonalize it and write \[\mathbf{A}= \mathbf{U}\mathbf{D}\mathbf{U}^\top\] where \(\mathbf{D}=\operatorname{diag}(\lambda_1(\mathbf{A}), \ldots, \lambda_n(\mathbf{A}))\) is a diagonal matrix of eigenvalues and \(\mathbf{U}\) is an orthogonal matrix consisting of unit eigenvectors.

Then \[\operatorname{Tr}(\mathbf{A}) = \operatorname{Tr}(\mathbf{U}\mathbf{D}\mathbf{U}^\top) = \operatorname{Tr}(\mathbf{U}^\top\mathbf{U}\mathbf{D}) = \operatorname{Tr}(\mathbf{D})=\sum_{i=1}^n \lambda_i(\mathbf{A})\] where the second equality follows as \(\operatorname{Tr}(\mathbf{A}\mathbf{B})= \operatorname{Tr}(\mathbf{B}\mathbf{A})\) and the third equality because \(\mathbf{U}^\top \mathbf{U}=\mathbf{I}\).

\[\det(\mathbf{A})=\det(\mathbf{U}\mathbf{D}\mathbf{U}^\top)=\det(\mathbf{U})\det(\mathbf{D})\det(\mathbf{U}^\top)=\det(\mathbf{D})=\prod_{i=1}^n\lambda_i(\mathbf{A})\]

where the third equality follows because we must have \(\det(\mathbf{U})=1\) as \(\mathbf{U}^\top \mathbf{U}=\mathbf{I}\) implies \(\det(\mathbf{U})^2 = \det(\mathbf{I})=1\). The fourth equality follows as \(\mathbf{D}\) is diagonal.

Question 6

If \(\mathbf{A}\) is a \(2\times 2\) symmetric matrix with \(\operatorname{Tr}(\mathbf{A})=3\) and \(\operatorname{det}(\mathbf{A})=1\), what are the eigenvalues of \(\mathbf{A}\)?

Solution

From the previous question we know \[\operatorname{Tr}(\mathbf{A})=\lambda_1+\lambda_2=3\mbox{ and } \operatorname{det}(\mathbf{A})=\lambda_1\lambda_2=1.\] Solving these two equations (with two unknowns) we find \[\lambda=\frac{3}{2}\pm \frac{\sqrt{5}}{2}\]

Question 7

Some topological questions:

Solution

\[ \begin{aligned} \operatorname{int}\left(\mathbb{R}_{+}^{n}\right) &=\mathbb{R}_{++}^{n} \\ \operatorname{int}(B[\mathbf{c}, r]) &= B(\mathbf{c}, r)\\% \quad\left(\bc \in \mathbb{R}^{n}, r \in \mathbb{R}_{++}\right) \\ \operatorname{int}([\mathbf{x}, {\mathbf{y}}]) &=\emptyset\\ \left(\mathbf{c} \in \mathbb{R}^{n}, r \in \mathbb{R}_{++}\right), \operatorname{bd}(B(\mathbf{c}, r)) &=\{\mathbf{x}:\|\mathbf{x}-\mathbf{c}\|=r\}\\ \left(\mathbf{c} \in \mathbb{R}^{n}, r \in \mathbb{R}_{++}\right), \operatorname{bd}(B[\mathbf{c}, r]) &=\{\mathbf{x}:\|\mathbf{x}-\mathbf{c}\|=r\}\\ \operatorname{bd}\left(\mathbb{R}_{++}^{n}\right) &=\{\mathbf{x}\in \mathbb{R}^n: x_i =0\mbox{ for some } i\}\\ \operatorname{bd}\left(\mathbb{R}_{+}^{n}\right) &=\{\mathbf{x}\in \mathbb{R}^n: x_i =0\mbox{ for some } i\}\\ \operatorname{bd}\left(\mathbb{R}^{n}\right) &=\emptyset\\ \operatorname{bd}\left(\Delta_{n}\right) &=\Delta_{n}\\ \operatorname{cl}\left(\mathbb{R}_{++}^{n}\right) &=\mathbb{R}_{+}^{n}\\ (\mathbf{x}\neq {\mathbf{y}}), \operatorname{cl}((\mathbf{x}, {\mathbf{y}})) &=[\mathbf{x}, {\mathbf{y}}] \end{aligned} \]

Question 8

Consider the function \(f(\mathbf{x}) =x^2 y^3 - 4xz\). What is the gradient of \(f\)? What is the directional derivative of \(f\) in the direction \((-1,2,0)\) at the point \(\mathbf{x}=(1,1,1)^\top\)? Thus find a linear and a quadratic approximation to the function \(f\) at the point \((1,1,1)\).

Solution

\[\nabla f(\mathbf{x}) =\begin{pmatrix} 2xy^3-4z\\ 3x^2y^2\\ -4x \end{pmatrix}\] The directional derivative in the direction \(\mathbf{d}=(-1, 2, 0)^\top\) at the point \(\mathbf{x}=(1,1,1)^\top\) is \[f'(\mathbf{x}; \mathbf{d}) = \nabla f(\mathbf{x})^\top \mathbf{d}= \begin{pmatrix} -2&3&-4\end{pmatrix}\begin{pmatrix}-1\\2\\0\end{pmatrix}=8.\]

Thus a linear approximation to \(f\) at the point \(\mathbf{x}=(1,1,1)^\top\) is \[\begin{align*}f(\mathbf{x}+\mathbf{h})&=f(\mathbf{x})+ \nabla f(\mathbf{x})^\top \mathbf{h}+o(\|\mathbf{h}\|)\\ &=-3+\begin{pmatrix}-2&3&-4\end{pmatrix}\mathbf{h}+o(\|\mathbf{h}\|).\end{align*}\]

To find a quadratic approximation, we need the Hessian, which is

\[\nabla^2 f(\mathbf{x})=\begin{pmatrix}2y^3&6xy^2&-4\\ 6xy^2&6x^2y&0\\ -4 & 0 &0\end{pmatrix}\] and so a quadratic approximation is \[\begin{align*}f(\mathbf{x}+\mathbf{h})&=f(\mathbf{x})+ \nabla f(\mathbf{x})^\top \mathbf{h}+\frac{1}{2}\mathbf{h}^\top\nabla^2 f(\mathbf{x}) \mathbf{h}+o(\|\mathbf{h}\|^2)\\ &=-3+\begin{pmatrix}-2&3&-4\end{pmatrix}\mathbf{h}+ \frac{1}{2}\mathbf{h}^\top \begin{pmatrix}2&6&-4\\ 6&6&0\\ -4 & 0 &0\end{pmatrix} \mathbf{h} +o(\|\mathbf{h}\|^2).\end{align*}\]

Question 9

Find direction \(\mathbf{d}\in \mathbb{R}^n\) with \(||\mathbf{d}||=1\) to minimize \(f'(\mathbf{x}; \mathbf{d})\). Hint: use Cauchy-Schwartz! Why might it be useful to know which direction has the steepest descent?

Solution

Cauchy-Schwartz gives \[|f'(\mathbf{x};\mathbf{d})| =|\nabla f(\mathbf{x})^\top \mathbf{d}| \leq \|\nabla f(\mathbf{x})\| \| \mathbf{d}\| = \|\nabla f(\mathbf{x})\|\] or equivalently,

\[- \|\nabla f(\mathbf{x})\|\leq f'(\mathbf{x};\mathbf{d})\leq \|\nabla f(\mathbf{x})\|\] If we set \(\mathbf{d}=-\frac{\nabla f(\mathbf{x})}{\|\nabla f(\mathbf{x})\|}\) then we have

\[f'(\mathbf{x};\mathbf{d})=-\nabla f(\mathbf{x})^\top \frac{\nabla f(\mathbf{x})}{\|\nabla f(\mathbf{x})\|}= - \frac{\|\nabla f(\mathbf{x})\|^2}{\|\nabla f(\mathbf{x})\|}=-\|\nabla f(\mathbf{x})\|.\] I.e., we achieve the lower bound given by Cauchy-Schwartz. Thus this choice minimizes \(f'(\mathbf{x};\mathbf{d})\).

This might be useful in optimization. If we are trying to minimize \(f(\mathbf{x})\) finding a direction \(\mathbf{d}\) that maximizes \(f(\mathbf{x})-f(\mathbf{x}+\mathbf{d})\) seems like a useful thing to do!

Question 10

Consider the following approximation to \(f(\mathbf{x})\) \[\tilde{f}(\mathbf{x}) = f({\mathbf{y}})+\nabla f({\mathbf{y}})^\top (\mathbf{x}-{\mathbf{y}}) + \frac{1}{2h} ||\mathbf{x}-{\mathbf{y}}||^2.\] where \(h>0\) is a constant. Find the value of \(\mathbf{x}\) that minimizes \(\tilde{f}(\mathbf{x})\). Given this result, can you think of an iterative approach for minimizing \(f(\mathbf{x})\)?

Repeat these steps using the quadratic approximation \[\tilde{f}(\mathbf{x}) = f({\mathbf{y}})+\nabla f({\mathbf{y}})^\top (\mathbf{x}-{\mathbf{y}}) + \frac{1}{2h} (\mathbf{x}-{\mathbf{y}})^\top \nabla^2 f({\mathbf{y}}) (\mathbf{x}-{\mathbf{y}})\]

Solution

Starting with \[\tilde{f}(\mathbf{x}) = f({\mathbf{y}})+\nabla f({\mathbf{y}})^\top (\mathbf{x}-{\mathbf{y}}) + \frac{1}{2h} ||\mathbf{x}-{\mathbf{y}}||^2\] if we differentiate we get \[\nabla \tilde{f}(\mathbf{x}) = \nabla f({\mathbf{y}})+\frac{1}{h}(\mathbf{x}-{\mathbf{y}})\] Setting this equal to zero and rearranging gives \[\mathbf{x}={\mathbf{y}}-h \nabla f({\mathbf{y}}).\]

As we will see, this is the gradient descent algorithm. To minimize \(f\) we could build a sequence of approximations and then minimize them. Using \(\tilde{f}\) as our approximation gives the iteration: \[\mathbf{x}_{k+1}=\mathbf{x}_k-h\nabla f(\mathbf{x}_k).\]

If instead, we use the approximation \[\tilde{f}(\mathbf{x}) = f({\mathbf{y}})+\nabla f({\mathbf{y}})^\top (\mathbf{x}-{\mathbf{y}}) + \frac{1}{2h} (\mathbf{x}-{\mathbf{y}})^\top \nabla^2 f({\mathbf{y}}) (\mathbf{x}-{\mathbf{y}})\] then \[\nabla \tilde{f}(\mathbf{x}) = \nabla f({\mathbf{y}})+\frac{1}{h}\nabla^2f({\mathbf{y}})(\mathbf{x}-{\mathbf{y}})\] If we again set this equal to zero and solve for \(\mathbf{x}\) we get the iteration \[\mathbf{x}= {\mathbf{y}}- h\nabla^2f({\mathbf{y}})^{-1}\nabla f({\mathbf{y}}).\] As we will see in week 5, this leads to Newtonâ€™s algorithm.
