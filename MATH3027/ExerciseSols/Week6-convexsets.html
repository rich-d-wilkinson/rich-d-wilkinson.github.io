Question 1

Intersection of convex sets is convex: Prove that if \(C_{i} \subseteq \mathbb{R}^{n}\) are convex sets for any \(i =1, \ldots, I\), then the set \(\bigcap_{i \in I} C_{i}\) is also convex.

Solution

These proofs all work in pretty much the same way:

Suppose \(x,y \in \bigcap_{i \in I} C_{i}\) and let \(\lambda \in [0,1]\). We need to show \(\lambda x+(1-\lambda)y \in \bigcap_{i \in I} C_{i}\).

Note that because \(x\) and \(y\) are in the intersection, we also have \(x, y\in C_i\) for all \(i\). The \(C_i\) are convex, and so \[\lambda x+(1-\lambda)y \in C_i \mbox{ for all }i.\]

Thus \(\lambda x+(1-\lambda)y \in\bigcap_{i \in I} C_{i}\)

Question 2

Let \(M \subseteq \mathbb{R}^{n}\) be a convex set and let \(\mathbf{A} \in \mathbb{R}^{m \times n}\). Prove that \[\begin{align*} \mathbf{A}(M)=\{\mathbf{A} \mathbf{x}: \mathbf{x} \in M\} \end{align*}\] is convex.

Solution

Suppose \(z, z' \in \mathbf{A}(M)\) and that \(\lambda\in[0,1]\). Then we must have \(z=\mathbf{A}x\) and \(z'=\mathbf{A}y\) for some \(x\) and \(y\). We have to show \(\lambda z + (1-\lambda)z\in \mathbf{A}(M)\).

We can see that \[\lambda z + (1-\lambda)z' = \lambda \mathbf{A}x+(1-\lambda)\mathbf{A}y =\mathbf{A}(\lambda x+(1-\lambda)y)\] But as \(M\) is convex we have that \(\lambda x+(1-\lambda)y\in M\), and thus \(\lambda z + (1-\lambda)z'\in\mathbf{A}(M)\) by definition of \(\mathbf{A}(M)\).

Question 3

Consider the four vectors \[\begin{align*} \mathbf{x}_{1}=\left(\begin{array}{l} 1 \\ 1 \end{array}\right), \mathbf{x}_{2}=\left(\begin{array}{l} 1 \\ 2 \end{array}\right), \mathbf{x}_{3}=\left(\begin{array}{l} 2 \\ 1 \end{array}\right), \mathbf{x}_{4}=\left(\begin{array}{l} 2 \\ 2 \end{array}\right) \end{align*}\] and let \(\mathbf{x} \in \operatorname{conv}\left(\left\{\mathbf{x}_{1}, \mathbf{x}_{2}, \mathbf{x}_{3}, \mathbf{x}_{4}\right\}\right)\) be given by \[\begin{align*} \mathbf{x}=\frac{1}{8} \mathbf{x}_{1}+\frac{1}{4} \mathbf{x}_{2}+\frac{1}{2} \mathbf{x}_{3}+\frac{1}{8} \mathbf{x}_{4}=\left(\begin{array}{c} \frac{13}{8} \\ \frac{11}{8} \end{array}\right) \end{align*}\] Is it possible to represent \(\mathbf{x}\) as a convex combination of no more than 3 vectors?

Solution

By the Carath'eodory representation theorem for convex hulls states that the convex hull of a set \(S\subset \mathbb{R}^n\) can be expressed as the convex hull of no more than \(n+1\) vectors from \(S\). So in this case, \(n=2\), so we know we must be able to write \(\mathbf{x}\) as the sum of 3 vectors.

A bit of thought will show that \[\mathbf{x}= \frac{3}{8}\mathbf{x}_1+\frac{1}{4}\mathbf{x}_2+\frac{1}{2}\mathbf{x}_3\] as \(\mathbf{x}_4 = 2\mathbf{x}_1\).

Question 4

Find all the basic feasible solutions of the linear system: \[\begin{align*} \begin{aligned} x_{1}+x_{2}+x_{3} &=6 \\ x_{2}+x_{4} &=3 \\ x_{1}, x_{2}, x_{3}, x_{4} & \geq 0. \end{aligned} \end{align*}\]

Solution

We can write this system as \[\begin{pmatrix}1&1&1&0\\ 0&1&0&1\end{pmatrix}\mathbf{x}= \begin{pmatrix} 6\\3\end{pmatrix} \mbox{ with }\mathbf{x}\geq \mathbf{0}.\]

From the definition of a BFS, we can see that we can have at most 2 elements of \(\mathbf{x}\) being positive.

  • If \(x_1, x_2>0\) this implies we must have \(x_3=x_4=0\) and so \(\mathbf{x}= \begin{pmatrix}3\\3\\0\\0\end{pmatrix}\).

  • If \(x_1, x_3>0\) this implies \(x_2=x_4=0\). Then there is no \(\mathbf{x}\) satisfying the constraints.

  • If \(x_1, x_4>0\) this implies we must have \(x_2=x_3=0\) and so \(\mathbf{x}= \begin{pmatrix}6\\0\\0\\3\end{pmatrix}\).

  • If \(x_2, x_3>0\) this implies we must have \(x_1=x_4=0\) and so \(\mathbf{x}= \begin{pmatrix}0\\3\\3\\0\end{pmatrix}\).

  • If \(x_2, x_4>0\) this implies we must have \(x_1=x_3=0\) and so \(x_2 =6\) and \(x_4=-3<0\). But this doesn’t satisfy the positivity constraint and so is not a feasible solution.

  • If \(x_3, x_4>0\) this implies we must have \(x_1=x_2=0\) and so \(\mathbf{x}= \begin{pmatrix}0\\0\\6\\3\end{pmatrix}\).

So in total there are 4 basic feasible solutions to this system.

Question 5

Consider the linear system \(\mathbf{A}\mathbf{x}={\mathbf{y}}\) where \(\mathbf{x}\geq \mathbf{0}\). If \(\mathbf{A}\) is \(m \times n\), then how many basic feasible solutions may there be to this system?

Solution

\(\mathbf{x}\) is a BFS of \(\mathbf{A}\mathbf{x}={\mathbf{y}}\) if the columns of \(\mathbf{A}\) corresponding to the positive entries in \(\mathbf{x}\) are linearly independent. Each column of \(\mathbf{A}\) is a vector in \(\mathbb{R}^m\) (as \(\mathbf{A}\) is \(m\times n\)), and we cannot have more than \(m\) linearly independent vectors in \(\mathbb{R}^m\). The number of ways of picking \(m\) columns from \(\mathbf{A}\) is \(n\choose m\), buts possible for BFS to have only \(m-1\) (or \(m-2\) etc) positive entries. Thus the total number of BFS cannot be larger than

\[\sum_{i=1}^m {n \choose i}\]

Even for small values of \(n\) and \(m\) this can still be a very large number. In this module, we don’t study algorithms for solving linear programmes - we just enumerate the BFS and look for the best. But this approach is only sensible in small problems.

Question 6

Find the extreme points of the system \[\begin{align*} \begin{aligned} x_{1}+x_{2}+x_{3} &=6 \\ x_{2}+x_{4} &=3 \\ x_{1}, x_{2}, x_{3}, x_{4} & \geq 0. \end{aligned} \end{align*}\]

Solution

This is the same system as in Q4. We know extreme points are just the basic feasible solutions, i.e., \[\begin{pmatrix}3\\3\\0\\0\end{pmatrix}, \begin{pmatrix}6\\0\\0\\3\end{pmatrix}, \begin{pmatrix}0\\3\\3\\0\end{pmatrix}, \begin{pmatrix}0\\0\\6\\3\end{pmatrix}. \]

Question 7

Show that the quad-over-lin function \[\begin{align*} f\left(x_{1}, x_{2}\right)=\frac{x_{1}^{2}}{x_{2}} \end{align*}\] defined over \(\mathbb{R} \times \mathbb{R}_{++}=\left\{\left(x_{1}, x_{2}\right): x_{2}>0\right\}\) is convex.

Solution

\[\nabla f = \begin{pmatrix} \frac{2x_1}{x_2}\\ -\frac{x_1^2}{x_2^2}\end{pmatrix}\quad \mbox{ and }\quad\nabla^2 f = 2\begin{pmatrix} \frac{1}{x_2} & -\frac{x_1}{x_2^2}\\ -\frac{x_1}{x_2^2} & \frac{x_1^2}{x_2^3}\end{pmatrix}\]

By the second-order characterization, \(f\) is convex if and only if \(\nabla^2 f(\mathbf{x}) \succeq 0\) for all \(\mathbf{x}\). In this case, we can see that \[\operatorname{Tr}(\nabla^2 f(x)) = \frac{2}{x_2}+\frac{2x_1^2}{x_2^3} >0\] and \[\det (\nabla^2 f(x)) = 4\frac{x_1^2}{x_2^2}-4\frac{x_1^2}{x_2^4}=0\] And so we can see that \(\nabla^2 f(\mathbf{x})\) is positive semi-definite for all \(\mathbf{x}\), and hence \(f\) is convex.

Question 8

Prove that the following functions are convex:

  • The generalized quad-over-lin function \[\begin{align*} g(\mathbf{x})=\frac{\|\mathbf{A} \mathbf{x}+\mathbf{b}\|^{2}}{\mathbf{c}^{\top} \mathbf{x}+d} \quad\left(\mathbf{A} \in \mathbb{R}^{m \times n}, \mathbf{b} \in \mathbb{R}^{m}, \mathbf{c} \in \mathbb{R}^{n}, d \in \mathbb{R}\right) \end{align*}\] over the domain \(D=\left\{\mathbf{x} \in \mathbb{R}^{n}: \mathbf{c}^{\top} \mathbf{x}+d>0\right\}\).

  • \(h(\mathbf{x})=e^{\|\mathbf{x}\|^{2}}\).

Solution

  • To prove \(g(\mathbf{x})=\frac{\|\mathbf{A} \mathbf{x}+\mathbf{b}\|^{2}}{\mathbf{c}^{\top} \mathbf{x}+d}\) is convex, we will first prove \[h({\mathbf{y}},t)=\frac{\|{\mathbf{y}}\|^2}{t}\] is convex. Note that \(h({\mathbf{y}},t) = \sum h_i({\mathbf{y}}, t)\) where \(h_i({\mathbf{y}}, t)=y_i^2/t\). By question 7 we know each \(h_i\) is convex, and thus the sum \(h\) is convex by the algebra of convex functions rules.

    Finally, note that \(g\) is obtained from \(h\) by an affine transformation of variables \[g(\mathbf{x}) = h(\mathbf{A}\mathbf{x}+\mathbf{b}, \mathbf{c}^\top\mathbf{x}+d)\] and so \(g\) is convex.

  • \(h(\mathbf{x})=e^{\|\mathbf{x}\|^{2}}\). We can write this as \[h(\mathbf{x}) = g(f(\mathbf{x}))\] where \(f(\mathbf{x})=\|\mathbf{x}\|^2\), which is a convex function, and \(g(x) = e^x\), which is convex, and non-decreasing. Thus \(h\) is convex by the algebraic rules from lectures.

Question 9

Let \(f_{1}, f_{2}, \ldots, f_{p}\) be convex functions over a convex set \(C \subseteq \mathbb{R}^{n}\). Prove that the sum function \(f_{1}+f_{2}+\ldots+f_{p}\) is also convex over \(C\).

Solution

Much like with convex sets, these proofs all work in a similar way:

Suppose \(\mathbf{x}, {\mathbf{y}}\in C\) and \(\lambda \in [0,1]\). Consider \[\begin{align*} (f_1+\ldots+f_p)(\lambda \mathbf{x}+(1-\lambda){\mathbf{y}}) &= f_1(\lambda \mathbf{x}+(1-\lambda){\mathbf{y}})+\ldots + f_p(\lambda \mathbf{x}+(1-\lambda){\mathbf{y}})\\ &\leq \lambda f_1(\mathbf{x})+(1-\lambda)f_1({\mathbf{y}})+\ldots + \lambda f_p(\mathbf{x})+(1-\lambda)f_p({\mathbf{y}})\\ &= \lambda (f_1+\ldots +f_p)(\mathbf{x})+(1-\lambda) (f_1+\ldots +f_p)({\mathbf{y}}) \end{align*}\] where the second line follows because all the \(f_i\) are convex.

Question 10

Point-Wise Maximum of Convex Functions Let \(f_{1}, f_{2}, \ldots, f_{p}: C \rightarrow \mathbb{R}\) be \(p\) convex functions over the convex set \(C \subseteq \mathbb{R}^{n}\). Prove that the maximum function \[\begin{align*} f(\mathbf{x}) \equiv \max _{i=1,2, \ldots, p}\left\{f_{i}(\mathbf{x})\right\} \end{align*}\] is convex over \(C\).

Solution

Let \(\mathbf{x}, {\mathbf{y}}\in C\) and \(\lambda \in [0,1]\). Then

\[\begin{align*} f(\lambda\mathbf{x}+(1-\lambda){\mathbf{y}}) &= \max f_i(\lambda\mathbf{x}+(1-\lambda){\mathbf{y}})\\ &\leq\max\left(\lambda f_i(\mathbf{x})+(1-\lambda)f_i({\mathbf{y}})\right)\\ &= \lambda \max f_i(\mathbf{x})+(1-\lambda)\max f_i({\mathbf{y}})\\ &= \lambda f(\mathbf{x})+(1-\lambda)f({\mathbf{y}}) \end{align*}\]

So \(f\) is convex.

Question 11

Proving the following functions are convex

  • \(f(\mathbf{x})=\max \left\{x_{1}, x_{2}, \ldots, x_{n}\right\}\).
  • For a given vector \(\mathbf{x}=\left(x_{1}, x_{2}, \ldots, x_{n}\right)^{\top} \in \mathbb{R}^{n},\) let \(x_{[i]}\) denote the \(i\) -th largest value in \(\mathbf{x}\). For any \(k \in\{1,2, \ldots, n\}\) let \[\begin{align*} h_{k}(\mathbf{x})=x_{[1]}+x_{[2]}+\ldots+x_{[k]}. \end{align*}\]
Solution

  • \(f\) is clearly convex by the result from Q1.

  • \[\begin{align*} h_{k}(\mathbf{x})&=x_{[1]}+x_{[2]}+\ldots+x_{[k]}\\ &=\max \{ x_{i_1}+x_{i_2}+\ldots+x_{i_k}: i_1, \ldots, i_k \in \{1, \ldots, n\} \mbox{ all different} \} \end{align*}\] Thus \(h_k\) is the maximum of linear functions (which are convex), and thus is convex.
