2.5 Probability of misclassification

Let \(p_{jk}\) denote the probability of allocating an observation to population \(\Pi_j\), when in fact it comes from \(\Pi_k\). Therefore \(p_{kk}\) is the probability of correctly classifying this observation and \(1-p_{kk}\) is the probability of misclassification.

One way of estimating \(p_{jk}\) is to consider the number of observations from the training data that are misclassified. For example, if \(n_k\) observations come from population \(k\) and \(n_{jk}\) is the number of observations from population \(k\) classified as from population \(j\), then \[ \hat{p}_{jk} = \frac{n_{jk}}{n_k} \] is an estimate of \(p_{jk}\).

When \(g=2\), \(\Pi_j\) is \(N_p({\boldsymbol{\mu}}_j, \boldsymbol{\Sigma})\) and we use the ML rule, we obtain an explicit expression for \(p_{12}\) and \(p_{21}\) as follows.

Recall that we allocate \(\mathbf z\) to \(\Pi_1\) if and only if \(U = \mathbf a^\top (\mathbf z-\mathbf h)>0\) where \(\mathbf a= \boldsymbol{\Sigma}^{-1} ({\boldsymbol{\mu}}_1 - {\boldsymbol{\mu}}_2)\) and \(\mathbf h= \frac{1}{2} ({\boldsymbol{\mu}}_1 + {\boldsymbol{\mu}}_2)\).

Suppose \(\mathbf z\) is from \(\Pi_2\). Then \(\mathbf z\sim N_p({\boldsymbol{\mu}}_2,\boldsymbol{\Sigma})\), so \[\begin{eqnarray*} E[U] &=& E[\mathbf a^\top (\mathbf z-\mathbf h)] = \mathbf a^\top (E[\mathbf z] -\mathbf h) = \mathbf a^\top ({\boldsymbol{\mu}}_2-\mathbf h); \\ \text{Var}(U) &=& \text{Var}(\mathbf a^\top \mathbf z- \mathbf a^\top \mathbf h) = \text{Var}(\mathbf a^\top \mathbf z) = \mathbf a^\top \boldsymbol{\Sigma}\mathbf a. \end{eqnarray*}\] Hence, when \(\mathbf z\) is from \(\Pi_2\), \(U \sim N(\mathbf a^\top ({\boldsymbol{\mu}}_2 - \mathbf h), \mathbf a^\top \boldsymbol{\Sigma}\mathbf a)\).

Define \(\Delta^2 = ({\boldsymbol{\mu}}_1-{\boldsymbol{\mu}}_2)^\top \boldsymbol{\Sigma}^{-1} ({\boldsymbol{\mu}}_1-{\boldsymbol{\mu}}_2)\), Then \[\begin{eqnarray*} \mathbf a^\top ({\boldsymbol{\mu}}_2-\mathbf h) &=& \mathbf a^\top \left({\boldsymbol{\mu}}_2 - \frac{1}{2}{\boldsymbol{\mu}}_1 - \frac{1}{2}{\boldsymbol{\mu}}_2 \right)= \frac{1}{2} \mathbf a^\top ({\boldsymbol{\mu}}_2 - {\boldsymbol{\mu}}_1) \\ &=& \frac{1}{2} ({\boldsymbol{\mu}}_1 - {\boldsymbol{\mu}}_2)^\top \boldsymbol{\Sigma}^{-1} ({\boldsymbol{\mu}}_2 - {\boldsymbol{\mu}}_1) \\ &=& -\frac{1}{2} ({\boldsymbol{\mu}}_1 - {\boldsymbol{\mu}}_2)^\top \boldsymbol{\Sigma}^{-1} ({\boldsymbol{\mu}}_1 - {\boldsymbol{\mu}}_2) = -\frac{1}{2}\Delta^2. \end{eqnarray*}\] and \[ \mathbf a^\top \boldsymbol{\Sigma}\mathbf a= ({\boldsymbol{\mu}}_1 - {\boldsymbol{\mu}}_2)^\top \boldsymbol{\Sigma}^{-1} \boldsymbol{\Sigma}\boldsymbol{\Sigma}^{-1} ({\boldsymbol{\mu}}_1 - {\boldsymbol{\mu}}_2) = \Delta ^2. \] Hence \(U \sim N( -\frac{1}{2}\Delta^2, \Delta^2)\) when \(\mathbf z\) is from \(\Pi_2\).

Now \(\mathbf z\) is allocated to \(\Pi_1\) if \(U>0\). The probability of this event when \(\mathbf z\) is in fact from \(\Pi_2\) is \[\begin{eqnarray*} p_{12} = P(U>0) &=& P \left(\frac{ U - (-\Delta^2/2) }{\Delta} > \frac{0 - (-\Delta^2/2) }{\Delta} \right)\\ &=& P \left(Z > \frac{\Delta^2}{2\Delta} = \frac{\Delta}{2} \bigg| Z \sim N(0,1) \right)\\ &=& P \left(Z < -\frac{\Delta}{2} \right) \end{eqnarray*}\] which can be found from statistical tables. A similar argument shows that \(p_{21}=p_{12}\).

If we are using the sample ML rule then we can replace \(\Delta^2\) with \[D^2 = (\bar{\mathbf x}_1 - \bar{\mathbf x}_2)^\top \widehat{\boldsymbol{\Sigma}}^{-1} (\bar{\mathbf x}_1 - \bar{\mathbf x}_2)\] and \(\hat{p}_{12} = P(Z<-D/2)\).
