Multivariate Statistics

5.1 Definition and Properties of the MVN

5.1.1 Basics

Definition 5.1 A random vector \(\bx=(x_1, \ldots , x_p)^\top\) has a \(p\)-dimensional MVN distribution if and only if \(\ba^\top \bx\) is a univariate normal random variable for all constant vectors \(\ba \in \mathbb{R}^p\).

In particular, note that the marginal distribution of each element of \(\bx\) has a uni-variate Gaussian distribution.

Notation: If \(\bx\in \mathbb{R}^p\) is MVN with mean \(\bmu\in \mathbb{R}^p\) and covariance matrix \(\bSigma \in \mathbb{R}^{p\times p}\) then we write \[ \bx \sim N_p (\bmu, \bSigma).\]



Definition 5.2 If the population covariance matrix \(\bSigma\) (\(p \times p\)) is positive definite (i.e. full rank), so that \(\bSigma^{-1}\) exists, then the probability density function (pdf) of the MVN distribution is given by \[ f(\bx) = \frac{1}{| 2 \pi \bSigma |^{1/2}} \exp \lb -\frac{1}{2}(\bx - \bmu)^\top \bSigma^{-1} (\bx - \bmu) \rb.\] Here, I’ve used the notation \(|\bA| = \det(\bA)\).

If \(p=1\), so that \(\bx = x\), \(\bmu = \mu\) and \(\bSigma = \sigma^2\), say, then the pdf simplifies to \[\begin{eqnarray*} f(\bx) &=& \frac{1}{|2 \pi \sigma^2|^{1/2}} \exp \lb -\frac{1}{2}(x - \mu) (\sigma^2)^{-1} (x - \mu) \rb \\ &=& \frac{1}{(2 \pi \sigma^2)^{1/2}} \exp \lb -\frac{1}{2 \sigma^2}(x - \mu)^2 \rb \end{eqnarray*}\] which is the familiar pdf of the univariate normal distribution \(N(\mu,\sigma^2)\).

If \(p>1\) and \(\bSigma = \operatorname{diag}(\sigma_1^2, \ldots, \sigma_p^2)\) then \[\begin{eqnarray*} f(\bx) &=& \frac{1}{((2 \pi)^{p}\prod_{i=1}^p \sigma_i^2)^{1/2}} \exp \lb -\frac{1}{2}(\bx - \bmu)^\top \bSigma^{-1}(\bx - \bmu) \rb \\ &=& \frac{1}{(2 \pi)^{p/2}\prod_{i=1}^p \sigma_i} \exp \lb -\frac{1}{2} \sum_{i=1}^p \frac{(x_i - \mu_i)^2}{\sigma_i^2} \rb \\ &=& \lb \frac{1}{\sqrt{2 \pi\sigma_1^2}} \exp \lb -\frac{1}{2\sigma_1^2} (x_1 - \mu_1)^2 \rb \rb \\ && \qquad \qquad \times \ldots \lb \frac{1}{\sqrt{2 \pi \sigma_p^2}} \exp \lb -\frac{1}{2\sigma_p^2} (x_p - \mu_p)^2 \rb \rb \end{eqnarray*}\] Thus, by the factorisation theorem for probability densities, the components of \(\bx\) have independent univariate normal distributions: \(x_i \sim N(\mu_i, \sigma_i^2)\).

If \(p=2\) we can plot \(f(\bx)\) using contour plots. Below, I’ve generated 1000 points from four different normal distributions using mean vectors \[\bmu_1=\bmu_3=\bmu_4=\begin{pmatrix}0 \\0 \\\end{pmatrix}, \quad \bmu_2=\begin{pmatrix}1 \\-1 \\\end{pmatrix}\] and covariance matrices \[\bSigma_1=\bSigma_2=\begin{pmatrix}1&0 \\0&1 \\\end{pmatrix}, \quad \bSigma_3=\begin{pmatrix}1&0 \\0&0.05 \\\end{pmatrix}, \quad \bSigma_4=\begin{pmatrix}1&0.9 \\0.9&1 \\\end{pmatrix}.\]

Note that the top left and bottom right plots have the same marginal distribtions for components \(x_1\) and \(x_2\), namely \[x_1 \sim N(0, 1)\quad x_2 \sim N(0, 1)\]

The contours on each plot are obtained by finding values of \(\bx\) for which \(f(\bx)=c\). The constant \(c\) is chosen so that the the shapes (which we’ll see below are ellipses) enclose 66% and 95% of the data.

5.1.2 Transformations

Proposition 5.1 If \(\bx \sim N_p(\bmu,\bSigma)\) then if \[\by = \bA \bx + \bc, \mbox{ where } \bA\in \mathbb{R}^{q \times p} \mbox{ and }\bc \in \mathbb{R}^q \mbox{ are constant},\] then \[\by \sim N_q(\bA \bmu + \bc, \bA \bSigma \bA^\top).\]

Proof. Let \(\bb\in \mathbb{R}^q\) be a constant vector. Then \[ \bb^\top \by = \bb^\top \bA \bx + \bb^\top \bc = \ba^\top \bx + \bb^\top \bc \] where \(\ba^\top = \bb^\top \bA\). Now \(\ba^\top \bx\) is univariate normal for all \(\ba\) since \(\bx\) is MVN. Therefore \(\bb^\top \by\) is univariate normal for all \(\bb\), so \(\by\) is MVN.

We can compute \(\BE(\by)=\bA\bmu+\bc\) and \(\var(\by)=\bA \bSigma \bA^\top\) using the properties listed in Section 0.3.



This implies that a linear transformation of a MVN random variable is also MVN. We can use this result to prove two important corollaries. The first corollary is useful for simulating data from a general MVN distribution.

Corollary 5.1 If \(\bx \sim N_p(\bzero,\bI_p)\) and \(\by = \bSigma^{1/2} \bx + \bmu\) then \[\by \sim N_p(\bmu,\bSigma).\]

Proof. Apply 5.1 with \(\bA = \bSigma^{1/2}\) and \(\bc = \bmu\). Therefore \[\BE(\by) = \bSigma^{1/2} \bzero_p + \bmu = \bmu\quad \mbox{and}\quad \var(\by) = \bSigma^{1/2} \bI_p \bSigma^{1/2} = \bSigma.\]

The second corollary says that any MVN random variable can be transformed into standard form.

Corollary 5.2 Suppose \(\bx \sim N_p(\bmu,\bSigma)\), where \(\bSigma\) has full rank. Then
\[\by = \bSigma^{-1/2}(\bx - \bmu) \sim N_p(\bzero,\bI_p).\]

Proof. Apply Proposition 5.1 with \(\bA = \bSigma^{-1/2}\) and \(\bc = - \bSigma^{-1/2} \bmu\).

5.1.3 Independence

Proposition 5.2 Two vectors \(\bx\in\mathbb{R}^p\) and \(\by\in\mathbb{R}^q\) which are jointly multivariate normal are independent if and only if they are uncorrelated, i.e. \(\cov(\bx,\by) = \bzero_{p,q}\).

Proof. The joint distribution of \(\bx\) and \(\by\) can always be factorized as \[p(\bx, \by)= p(\bx)p(\by |\bx).\] If the conditional distribution for \(\by\) given \(\bx\) does not depend upon \(\bx\), i.e., if \(p(\by | \bx)=p(\by)\), then \(\by\) and \(\bx\) are independent.

Suppose \(\bx\sim N_p(\bmu_\bx, \bSigma_{\bx,\bx})\) and \(\by\sim N_p(\bmu_\by, \bSigma_{\by,\by})\) are jointly normally distributed and that they are uncorrelated. Thus we can write \[\begin{pmatrix}\bx\\ \by\end{pmatrix}\sim N_{p+q}\left(\bmu, \bSigma\right) \] where \[\bmu = \begin{pmatrix}\bmu_\bx\\ \bmu_{\by}\end{pmatrix}\quad \bSigma = \begin{pmatrix} \bSigma_{\bx, \bx} &\bzero_{p,q}\\ \bzero_{q,p} & \bSigma_{\bz, \bz}\end{pmatrix}.\]

Now \[\begin{align*} p(\by|\bx)&=\frac{p(\bx, \by)}{p(\bx)}\\ &\propto \frac{\exp\left(-\frac{1}{2}(\bx-\bmu_\bx)^\top \bSigma_{\bx,\bx}(\bx-\bmu_\bx) -\frac{1}{2}(\by-\bmu_\by)^\top \bSigma_{\by,\by}(\by-\bmu_\by)\right)}{\exp\left(-\frac{1}{2}(\bx-\bmu_\bx)^\top \bSigma_{\bx,\bx}(\bx-\bmu_\bx)\right)} \\ &\propto \exp\left(-\frac{1}{2}(\by-\bmu_\by)^\top \bSigma_{\by,\by}(\by-\bmu_\by)\right) \\ &\propto p(\by) \end{align*}\] So \(p(\by|\bx)=p(\by)\), i.e. \(p(\by |\bx)\) is not a function of \(\bx\), and thus \(\bx\) and \(\by\) are independent.



Proposition 5.2 means that zero correlation implies independence for the MVN distribution. This is not true in general for other distributions.

Note: It is important that \(\bx\) and \(\by\) are jointly multivariate normal. For example, suppose \(x \sim N(0, 1)\). Let \[z=\begin{cases} 1 \mbox{ with probability } \frac{1}{2}\\ -1 \mbox{ otherwise} \end{cases} \] and let \(y=zx\). Then clearly \(y\) is also a normal random variable: \(y \sim N(0,1)\). In addition, note that \[\cov(x,y)= \BE(xy)= \BE(x^2)\BE(z)=0\] so that \(x\) and \(y\) are uncorrelated.

However, \(x\) and \(y\) are clearly not independent: if you tell me \(x\), then I know \(y=x\) or \(y=-x\).

5.1.4 Confidence ellipses

The contours in the plots of the bivariate normal samples shown above are ellipses. They were defined to lines of consant density, i.e., by \(f(\bx)=c\), which implies

\[\begin{equation} (\bx - \bmu)^\top \bSigma^{-1} (\bx - \bmu)=c' \tag{5.1} \end{equation}\] for some constant \(c'\). To see that this is the equation of an ellipse, note that a standard ellipse in \(\mathbb{R}^2\) is given by the equation \[\begin{equation} \frac{x^2}{a^2}+\frac{y^2}{b^2}=1 \quad (a, b>0) \tag{5.2} \end{equation}\] and recall that a standard ellipse has axes of symmetry given by the \(x\)-axis and \(y\)-axis (if \(a>b\), the \(x\)-axis is the major axis, and the \(y\)-axis the minor axis). For example, \(a=10, b=3\) gives the ellipse:

If we define \({\mathbf A}=\left( \begin{array}{cc} a^2&0\\ 0&b^2 \end{array} \right)\) and write \({\mathbf x}=\binom{x}{y}\), then Equation (5.2) can be written in the form \[ \bx^\top {\mathbf A}^{-1}\bx =c'. \] To shift the centre of the ellipse from the origin to the point \(\bmu\) we modify the equation to be \[ (\bx-\bmu)^\top {\mathbf A}^{-1}(\bx-\bmu) =c'.\]

What if instead of using a diagonal matrix \(\bA\), we use a non-diagonal matrix \(\bSigma\) as in Equation (5.1)? If \(\bSigma\) has spectral decomposition \(\bSigma = \bV \bLambda\bV^\top\), then \[\begin{align*} (\bx-\bmu)^\top {\bSigma}^{-1}(\bx-\bmu) &= (\bx-\bmu)^\top \bV \bLambda^{-1}\bV^\top(\bx-\bmu) \\ &= \by^\top \bLambda^{-1}\by \\ &= \frac{y_1^2}{\lambda_1}+\frac{y_2^2}{\lambda_2} \end{align*}\] where \(\bLambda=\operatorname{diag}(\lambda_1, \lambda_2)\) is a diagonal matrix of eigenvalues, and \(\by= \bV^\top (\bx-\bu)\). Because \(\bV\) is an orthogonal matrix (a rotation), we can see that this is the equation of a standard ellipse when using the eigenvectors as the coordinate system. Or in other words, it is an ellipse with major axis given by the first eigenvector, and minor axis given by the second eigenvector, centered around \(\bmu\).

Analogous results for ellipsoids and quadratic forms hold in three and higher dimensions.

The term \((\bx-\bmu)^\top \bSigma^{-1} (\bx-\bmu)\) will be important later in the chapter. The following proposition gives its distribution:

Proposition 5.3 If \(\bx \sim N_p(\bmu, \bSigma)\) and \(\bSigma\) is positive definite then \[(\bx-\bmu)^\top \bSigma^{-1} (\bx-\bmu) \sim \chi_p^2.\]

Proof. Define \(\by = \bSigma^{-1/2} (\bx-\bmu)\) so \[\begin{align*} (\bx-\bmu)^\top \bSigma^{-1} (\bx-\bmu) &= \lb \bSigma^{-1/2} (\bx-\bmu) \rb^\top \lb \bSigma^{-1/2} (\bx-\bmu) \rb \\ &= \by^\top \by = \sum_{i=1}^p y_i^2 \end{align*}\] By Corollary 5.2, \(\by \sim N_p (\bzero, \bI_p)\), and so the components of \(\by\) have independent univariate normal distributions with mean 0 and variance 1. Recall from univariate statistics that if \(z \sim N(0,1)\) then \(z^2 \sim \chi^2_1\) and if \(z_1, \ldots, z_n\) are iid \(N(0,1)\) then \(\sum_{i=1}^n z_i^2 \sim \chi_n^2\). It therefore follows that \[\sum_{i=1}^p y_i^2 \sim \chi^2_p.\]



Proposition 5.3 means we can calculate the probability \[\BP\left((\bx-\bmu)^\top \bSigma^{-1} (\bx-\bmu)<k\right),\] which is the probability of \(\bx\) lying within a particular ellipsoid. We often use this to draw confidence ellipses, which are ellipses that we expect to contain some specified proportion of the random samples (95% say).

5.1.5 Sampling results for the MVN

In this section we present two important results which are natural generalisations of what happens in the univariate case.

Proposition 5.4 If \(\bx_1, \ldots, \bx_n\) is an IID random sample from \(N_p(\bmu, \bSigma)\), then the sample mean and sample variance matrix \[\bar{\bx} = \frac{1}{n} \sum_{i=1}^n \bx_i, \quad \bS = \frac{1}{n} \sum_{i=1}^n (\bx_i - \bar{\bx})(\bx_i-\bar{\bx})^\top\] are independent.

Proof. From Proposition 5.1 we can see that if \(\bx_1, \ldots, \bx_n \sim N_p(\bmu, \bSigma)\) then \(\bar{\bx} \sim N_p (\bmu, n^{-1}\bSigma)\). Let \(\by_i = \bx_i -\bar{\bx}\). Then \[\begin{align*} \cov(\bar{\bx},\by_i)&=\cov(\bar{\bx}, \bx_i -\bar{\bx})\\ &=\cov(\bar{\bx}, \bx_i) - \cov(\bar{\bx}, \bar{\bx})\\ &=n^{-1}\sum_{j=1}^n \left \{\BE[(\bx_j -\bmu)(\bx_i-\bmu)^\top]\right \}\\ & \qquad \qquad -\BE[(\bar{\bx}-\bmu)(\bar{\bx}-\bmu)^\top]\\ &=n^{-1}\bSigma - n^{-1}\bSigma\\ &= {\mathbf 0}_{p,p}. \end{align*}\] Thus Proposition 5.2 gives that \(\bar{\bx}\) and \(\by_i\) are independent, and therefore \(\bar{\bx}\) and \[ \bS=\frac{1}{n}\sum_{i=1}^n \by_i \by_i^\top =n^{-1}\sum_{i=1}^n (\bx_i -\bar{\bx})(\bx_i -\bar{\bx})^\top \] are independent.



Recall from above that if \(\bx_1, \ldots, \bx_n\) is a random sample from \(N_p(\bmu, \bSigma)\) then \[\bar{\bx} \sim N_p (\bmu, \frac{1}{n}\bSigma).\] This result is also approximately true for large samples from non-normal distributions, as is now stated in the multivariate central limit theorem.

Proposition 5.5 Central limit theorem Let \(\bx_1, \bx_2, \ldots \in \mathbb{R}^p\) be a sample of independent and identically distributed random vectors from a distribution with mean \(\bmu\) and finite variance matrix \(\bSigma\). Then asymptotically as \(n \rightarrow \infty\), \(\sqrt{n}(\bar{\bx}-\bmu)\) converges in distribution to \(N_p ({\mathbf 0}_p, \bSigma )\).
