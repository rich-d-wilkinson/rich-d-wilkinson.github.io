Definition 0.1 For a statistical model \(p(x\mid \theta)\), a statistic \(\hat{\theta}\equiv\hat{\theta}(X)\) is said to be an unbiased estimator of \(\theta\) if \({\mathbb{E}}_{x\mid \theta}(\hat{\theta}(X)) = \theta\) for all \(\theta\).

I.e., if data are generated with parameter value \(\theta\), then an estimator \(\hat{\theta}\) is an unbiased estimator of \(\theta\) if its expected value matches the true value (the value used to generate the data). This concept readily transfers to the multivariate context.

Proposition 0.1 Let \(\mathbf x_1, \ldots \mathbf x_n\) be independent and identically distributed (i.i.d.), sampled from a population with mean \({\boldsymbol{\mu}}\) and covariance matrix \(\boldsymbol{\Sigma}\). If \(\bar{\mathbf x}\) and \(\mathbf S\) are the sample mean and covariance matrix respectively, then

  1. \({\mathbb{E}}(\bar{\mathbf x}) = {\boldsymbol{\mu}}\).

  2. \({\mathbb{V}\operatorname{ar}}(\bar{\mathbf x}) = {\displaystyle\frac{1}{n}} \boldsymbol{\Sigma}\).

  3. \({\mathbb{E}}(\mathbf S) = {\displaystyle\frac{n-1}{n}} \boldsymbol{\Sigma}\).

Proof. Part 1 By the linearity of expectation, \[{\mathbb{E}}(\bar{\mathbf x}) = {\mathbb{E}}\left(\frac{1}{n} \sum_{i=1}^n \mathbf x_i \right)= \frac{1}{n} \sum_{i=1}^n {\mathbb{E}}(\mathbf x_i) = \frac{1}{n} n {\boldsymbol{\mu}}= {\boldsymbol{\mu}}.\] Part 2 We have \[\begin{eqnarray*} {\mathbb{V}\operatorname{ar}}(\bar{\mathbf x}) &=& {\mathbb{V}\operatorname{ar}}\left(\frac{1}{n} \sum_{i=1}^n \mathbf x_i \right)\\ &=& \sum_{i,j=1}^n {\mathbb{C}\operatorname{ov}}\left (\frac{1}{n}\mathbf x_i, \frac{1}{n}\mathbf x_j \right)\\ &=& \sum_{i=1}^n {\mathbb{V}\operatorname{ar}}\left(\frac{1}{n} \mathbf x_i \right)+ \sum_{i \neq j} {\mathbb{C}\operatorname{ov}}\left(\frac{1}{n} \mathbf x_i, \frac{1}{n} \mathbf x_j \right)\\ &=& \frac{1}{n^2} \left(\sum_{i=1}^n {\mathbb{V}\operatorname{ar}}(\mathbf x_i) + \sum_{i \neq j} {\mathbb{C}\operatorname{ov}}(\mathbf x_i,\mathbf x_j) \right)\\ &=& \frac{1}{n^2} \left(\sum_{i=1}^n {\mathbb{V}\operatorname{ar}}(\mathbf x_i) \right)\mbox{ as } \mathbf x_i \perp \!\!\! \perp\mathbf x_j \mbox{ for } i\not = j\\ &=& \frac{1}{n^2} n \boldsymbol{\Sigma}\\ &=& \frac{1}{n} \boldsymbol{\Sigma}. \end{eqnarray*}\]

Part 3 From the definition of the sample covariance, \[\begin{eqnarray*} \mathbf S&=& \frac{1}{n} \sum_{i=1}^n (\mathbf x_i -\bar{\mathbf x})( \mathbf x_i - \bar{\mathbf x})^\top \\ &=& \frac{1}{n} \sum_{i=1}^n (\mathbf x_i - {\boldsymbol{\mu}}+ {\boldsymbol{\mu}}-\bar{\mathbf x})(\mathbf x_i - {\boldsymbol{\mu}}+{\boldsymbol{\mu}}-\bar{\mathbf x})^\top \\ &=& \frac{1}{n} \sum_{i=1}^n \bigg \{(\mathbf x_i - {\boldsymbol{\mu}})(\mathbf x_i - {\boldsymbol{\mu}})^\top +(\bar{\mathbf x}-{\boldsymbol{\mu}})(\bar{\mathbf x}-{\boldsymbol{\mu}})^\top \\ && \qquad -(\mathbf x_i-{\boldsymbol{\mu}})(\bar{\mathbf x}-{\boldsymbol{\mu}})^\top -(\bar{\mathbf x}-{\boldsymbol{\mu}})(\mathbf x_i-{\boldsymbol{\mu}})^\top \bigg \}\\ &=& \frac{1}{n} \left \{\sum_{i=1}^n (\mathbf x_i - {\boldsymbol{\mu}})(\mathbf x_i - {\boldsymbol{\mu}})^\top \right \}- (\bar{\mathbf x}-{\boldsymbol{\mu}})(\bar{\mathbf x}-{\boldsymbol{\mu}})^\top \end{eqnarray*}\] as \(\frac{1}{n}\sum (\bar{\mathbf x}-{\boldsymbol{\mu}})(\mathbf x_i-{\boldsymbol{\mu}})^\top = \sum (\bar{\mathbf x}-{\boldsymbol{\mu}})(\bar{\mathbf x} -{\boldsymbol{\mu}})^\top\). Since \({\mathbb{E}}(\bar{\mathbf x})={\boldsymbol{\mu}}\) and \({\mathbb{V}\operatorname{ar}}(\bar{\mathbf x})=n^{-1}\boldsymbol{\Sigma}\), it follows that

\[\begin{eqnarray*} {\mathbb{E}}(\mathbf S)&=&\left \{\frac{1}{n}\sum_{i=1}^n {\mathbb{E}}\left [(\mathbf x_i - {\boldsymbol{\mu}})(\mathbf x_i - {\boldsymbol{\mu}})^\top \right ]\right \} -{\mathbb{V}\operatorname{ar}}(\bar{\mathbf x})\\ &=&{\mathbb{E}}\left \{(\mathbf x_1 - {\boldsymbol{\mu}})(\mathbf x_1 - {\boldsymbol{\mu}})^\top \right \} -{\mathbb{V}\operatorname{ar}}(\bar{\mathbf x})\\ &=& {\mathbb{V}\operatorname{ar}}(\mathbf x_1)-{\mathbb{V}\operatorname{ar}}(\bar{\mathbf x}) \\ &=& \boldsymbol{\Sigma}- \frac{1}{n}\boldsymbol{\Sigma}\\ &=& \frac{n-1}{n} \boldsymbol{\Sigma}, \end{eqnarray*}\]

which completes the proof.

An implication of this theorem is that \(\bar{\mathbf x}\) is an unbiased estimator for \({\boldsymbol{\mu}}\) but that \(\mathbf S\) is a biased estimator of \(\boldsymbol{\Sigma}\). Note, however, that \({\displaystyle\frac{n}{n-1}} \mathbf S\) is an unbiased estimator of \(\boldsymbol{\Sigma}\), i.e. \[ \frac{n}{n-1}{\mathbb{E}}[\mathbf S]=\boldsymbol{\Sigma}. \]

0.1.2 asdfa

In Part IV, we focus on different methods of classification, i.e. allocating the observations in a sample to different subsets (or groups). In Chapter 1, we focus on an approach called discriminant analysis, in which we have a training sample available, and we use this training sample to set up a suitable classification rule. An important type of situation where discriminant analysis is used is in screening tests. Here, several variables may be measured on each of a number of individuals, and we want to decide whether each individual is ‘’negative’‘, in which case no further investigations are required, or’‘positive’’, in which case further tests are required.

In Chapter ??, we consider an alternative approach, known as cluster analysis, in which we allocate the observations into clusters (or similar subsets). Here, a training sample is not available and typically the number of clusters will not be known in advance. The idea is to form clusters in such a way that experimental units within clusters are as similar as possible, in a suitable sense, and experimental units in different clusters are as dissimilar as possible.
