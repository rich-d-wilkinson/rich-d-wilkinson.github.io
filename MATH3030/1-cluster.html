Chapter 1 Cluster Analysis

Discriminant analysis, covered in Chapter ??, is a supervised learning method: in order to train the classifier we had access to both the input \(\mathbf x\) and the label for that case (what group it belonged to). This chapter focusses on cluster analysis, which is an unsupervised learning method: we have to train the model using only data on the input \(\mathbf x\). We are not given access to the cluster labels.

We shall limit ourselves to two topics within cluster analysis:

  • likelihood-based clustering which, as we will see, includes the widely-used method \(K\)-means clustering as a special case; and
  • hierarchical clustering methods.
